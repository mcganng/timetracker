Restore Instructions

To restore a backup:

  1. Download the backup file (.tar.gz) to your local machine
  2. Transfer the file to the target server
  3. Extract the backup: 
    tar -xzf timetracker_backup_YYYYMMDD_HHMMSS.tar.gz
  4. Run the restore script: 
    cd timetracker_backup_YYYYMMDD_HHMMSS
sudo bash restore.sh

Warning: Restoring a backup will overwrite all current data in the database and application files. Always create a backup of the current system before restoring!
