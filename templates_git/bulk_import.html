How to Use
Required Format in Savant

Headers (must be in this exact order):

Type | Job Number | Job Task No | Description | Monday | Tuesday | Wednesday | Thursday | Friday | Saturday | Sunday | Reason | Approved/Rejected | Person Responsible | Manager | Job Description | Customer Name | Total Hours | Status
From SAVANT:
  1. Log into Savant
  2. Click on your TE Section
  3. On a row with status "Historical", click the Pencil
  4. Under the Type column, click the first cell (Job or Absence)
  5. Press Ctrl+A to highlight all cells
  6. Press Ctrl+C to copy
To TIME TRACKER:
  1. If new project exists: Add it in Projects tab first
  2. Click Bulk Import tab
  3. Set Monday Date to match Savant week
  4. Paste copied data in the text box
  5. Click "Process Import"
